# Campaigns Module
The Campaigns module allows users to create, schedule, and analyze email campaigns.

Feature Summary:
- Email send limit varies by plan:
  - Free: 5,000 emails/month
  - Pro: 25,000 emails/month
  - Enterprise: 100,000 emails/month
- Built-in template designer and A/B testing.
- Suppression lists for unsubscribed contacts.
- Email deliverability analytics with bounce tracking.

To improve deliverability, maintain list hygiene and warm up new sender domains gradually.
