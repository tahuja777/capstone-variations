# Security & Data Residency FAQ
NovaCRM encrypts all data in transit (TLS 1.3) and at rest (AES-256).

Regions:
- India: Mumbai
- US: Virginia
- EU: Frankfurt

Admins can choose data residency at account creation. Backups occur daily and are retained for 30 days.

User data is never shared with third parties without consent, and access is logged for audit compliance.
