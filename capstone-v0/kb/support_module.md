# Support Module
The Support module provides customer service ticket management integrated with SLAs.

Key Details:
- Ticket priorities: Low, Medium, High, Critical.
- SLA targets: 24h (Low), 12h (Medium), 4h (High), 1h (Critical).
- Escalation workflow auto-triggers when SLA is breached.
- Integration with email and chat channels.
- Managers can review SLA compliance and agent performance in the dashboard.

Support analytics help identify top issue categories and reduce resolution time.
